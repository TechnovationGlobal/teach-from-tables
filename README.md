# teach-from-tables
Simple machine learning model builder that uses KNN algorithm to train a model from an uploaded spreadsheet. SImple preprocessing steps outlined for users. Meant as an analogous way for students to train a ML model for text and numerical data.

1. Users upload a spreadsheet
2. Users are guided to review the content, check duplicates, missing values, outliers and fix
3. Users click button to train model, using KNN algorithm
4. Users can then test their model in the user window.
5. They can then move on to exporting. Options are to export a HTML/Javacript package with model, a Streamlist package with model, or just the model. For either package, there are instructions for how to run it in the readme. The file is just a starter project with a simple interface to run the prediction model.
